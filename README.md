# DaysOfJFE

## 简述

最新垃圾代码力作

期末大作业要求写个小游戏，所以就做了这么一个**视觉小说**，不过跟我最初白日梦有着较大的差距，怎么说，有一些失败

我也没怎么学过 C++，问就是 AI 教的，你问我什么是模板，不好意思，只知道用。一开始学校要求用 [EasyX](https://easyx.cn/)，不过 EaxyX 不带抗锯齿，所以就直接使用 GDI+ 进行绘图

自己动手实现了一个简陋的 GUI 框架，称得上是地基是 WindowManager、WindowPanel、Controls、IGlobalEvent，入口在 main.cpp，会初始化 WindowManager 和一个 IGlobalEvent（提供公共事件的），然后添加继承了 WindowPanel 的窗口，选择要绘制的窗口就 OK，里面还有一些奇奇怪怪的"Hook"

除此之外还有一个简易的脚本系统，附带有一个简易的命令系统，有切换背景、播放音频（有 VOICE、BGM、SFX 三个声段）、角色高亮/隐藏、显示标题、跳转场景等功能。其中播放音频使用的是[BASS](https://www.un4seen.com/)这个库

里面的一些功能可能是用了一些奇奇怪怪的方法实现的，大佬要喷轻点

我个人比较反对 AI 完全参与到美术创造当中，所以这个项目中没有配音、背景图也只是手机拍的照片然后随手调了滤镜

对话框是对着苍之彼方四重奏的对话框抄的，其实我很喜欢「苍之彼方四重奏」「CloverDays」的艺术效果的，只可惜能力有限，而且剧本也配不上

剧本里除了第 5 章我都不满意，但是，写不出呀

可以的话，其实我还是想以「会长是女仆大人」「路人女主的 CTF 养成方法」「这个网安协会大有问题」「中二病也要打 CTF」等等标题满足我的兴趣的

## 后言

其实我一开始是想做一个私货满满的**视觉小说**的，不过嘛，真让我写起来，又是另外一回事。

写不出「CLANND」「LB」「AB」那样有感人的剧情

写不出「请问您今天要来点兔子吗」「幸运星」「珈百璃的堕落」「GJ 部」那样的日常

写不出「学生会的一己之见」「这个勇者明明超强却过分慎重」那样搞笑又有些许心酸的故事

写不出「擅长捉弄的高木同学」「关于我无意间被隔壁的天使逐渐养成废人这件事」那样甜甜的恋情

写不出「龙与虎」「我的青春恋爱物语果然有问题」「我的妹妹不可能这么可爱」那样的高中物语

写不出「苍之彼方四重奏」那样轰轰烈烈的空竟

没有常磐华乃、和泉纱雾的画技，没有羽岛伊月的写作能力……

所谓视觉小说，应该是能给人丰富的情感体验，只可惜，这个作品更多称为“一次作业”
